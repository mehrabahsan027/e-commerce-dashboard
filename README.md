
# Title:  E-commerce Analytics Dashboard

A comprehensive analytics dashboard for e-commerce businesses that provides real-time insights into key performance metrics including active users, monthly revenue, orders, and conversion rates with interactive visualizations.


## Features

--Active Users Tracking - Monitor real-time active users on your platform

--Revenue Analytics - Visualize monthly revenue trends with interactive charts

--Order Management - Track and analyze order statistics

--Conversion Rate Metrics - Monitor and optimize conversion performance

--Interactive Charts - Dynamic data visualization using Recharts

--Responsive Design - Fully responsive UI built with Tailwind CSS
